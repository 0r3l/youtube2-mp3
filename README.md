# Youtube to mp3
## Installation
- Run `npm i ytdl -g`
- Add `ffmpeg` binary to PATH from [here](https://www.ffmpeg.org/download.html#get-sources)

## Usage
Run `./run.bash "<id-of-youtube-video>" "<name-of-file-without-extenstion>"`
